# Diffusion Chat 페이지

이미지 생성을 위한 채팅 인터페이스입니다.

## 기능

1. **텍스트 입력**: 프롬프트와 네거티브 프롬프트 입력
2. **이미지 생성**: Stable Diffusion 모델을 사용한 이미지 생성
3. **고급 설정**: 너비, 높이, 스텝 수, guidance scale, seed 조정
4. **이미지 다운로드**: 생성된 이미지 다운로드
5. **생성 히스토리**: 대화 형태로 생성 이력 확인

## API 연동

- **서비스 URL**: `http://localhost:8000` (환경 변수로 변경 가능)
- **엔드포인트**: `/api/v1/generate`
- **메서드**: POST

## 환경 변수 설정

`.env.local` 파일에 다음을 추가:

```env
NEXT_PUBLIC_DIFFUSION_API_URL=http://localhost:8000
```

프로덕션 환경에서는 실제 서버 URL로 변경하세요.

## 사용 방법

1. 프롬프트 입력: 생성하고 싶은 이미지를 설명
2. (선택) 네거티브 프롬프트: 제외하고 싶은 요소 입력
3. (선택) 고급 설정에서 파라미터 조정
4. "이미지 생성" 버튼 클릭
5. 생성된 이미지 확인 및 다운로드
